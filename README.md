My submission for the 6.1D distinction task.
This learning app utilizes Ollama to generate hints for questions or explanations for why answers are aren't correct. The Ollama interfacing code is all built off of Naveet's demonstration code. The app also utilises a Room database to storte user accounts. The currently logged in user is stored through a shared preference. Questions are stored in a list of question classes which contains information about the question text, the answers and the correct answers. In future, these questions could be automatically generated, I did experiment with this and had success using the following prompt.
Generate a multiple choice question in this exact format QUESTION: {question} ANSWER0: {1st multiple choice option} ANSWER1: {2nd multiple choice option} ANSWER2: {3rd multiple choice option} ANSWER3: {4th multiple choice option} CORRECTANSWER: {the INT index of the correct answer}
